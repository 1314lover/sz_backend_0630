# Army training system
## Project desciption
建设一个面向军队训练场景的综合性训练管理系统，实现训练过程的数字化、智能化管理，整合物联网设备。
这个系统是给军队官兵训练时使用：涉及到多场景，多地方，不同参训人员进行训练评估的的训练管理系统。
训练成员在如下几个区域中进行：幽闭训练仓；高空训练台；临战训练区；搜救训练区；烟雾迷笼训练区；逃生训练区；综合管理区；这些区域对应执行不同的训练任务。
幽闭训练仓：参训人员在狭窄且注满水的封闭的仓室内从外面爬进去再爬出来，这个过程中需要对参训人员的心理和身体变化进行记录且评分。
高空训练台：参训人员在较高的区域尽心作业训练，对人员体征数据：如心率，呼吸率，hrv等进行记录上传，显示在系统的页面上。
搜救训练区：参训人员在指定的地方进行搜救任务，这个过程中可能会遇到假人扮演的道具，火花机喷出的烟火，电灯的诡异闪烁，烟雾踏板产生的烟等。
逃生训练区：参训人员在训练逃生区逃出来，会有时间的规定。
综合管理区：在综合管理平台上可以看到训练的情况和数据。

以上人员体征数据都是靠手环，心电贴，手机，NFC卡进行上传，传到后台就会采用相应的算法进行计算给出得分结果；实时数据是上传到tdengine数据库中

## System Composition
- **信息支撑系统**：面向系统管理员，负责基础配置（物联网数据的配置，角色，权限配置给到综合管理系统去使用）系统管理
  - 用户管理、角色管理、权限管理
  - 物联网设备配置
  - 系统监控
  - 日志监控
- **综合管理系统**：面向参训人员，教员的使用系统
  - 组训计划（组训计划中有人员，科目，时间，科目课程的设置）
  - 分级导调 (科目导调，参训人员进行科目训练)
  - 训练评估 (训练过程的监控评估，结果分数的评判)
